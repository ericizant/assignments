How Much Money Is There in the World?

Published on 11/11/2017

Written by Eric Izant

Ever wonder how much money there is in the world?

The answer is complicated, which you might expect, but not because of the difficulty of tallying up all the rather large numbers. Rather, it’s more about which parameters are used to define “money.”

“The amount of money that exists changes depending on how we define it. The more abstract definition of money we use, the higher the number is,” said Jeff Desjardins, an editor of Visual Capitalist, who put together an infographic to answer this question back in 2015 and recently updated it as money continues to take on a different meaning. “Are the abstractions created by Central Banks really money? What about gold, bitcoins, or other hard assets?” he asks.
